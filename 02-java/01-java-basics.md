# \u2615 Java Basics

> Foundation of Java programming for backend developers

## What is Java?

Java is a general-purpose, object-oriented programming language designed with the principle of "Write Once, Run Anywhere" (WORA). It runs on the Java Virtual Machine (JVM).

## Core Concepts

### JDK, JRE, and JVM

- **JVM**: Runtime environment that executes Java bytecode
- **JRE**: JVM + libraries needed to run Java programs
- **JDK**: JRE + compiler + tools for development

## Data Types

### Primitive Data Types

```java
byte age = 25;              // 1 byte
short port = 8080;          // 2 bytes
int count = 1000;           // 4 bytes
long distance = 9999999L;   // 8 bytes
float price = 99.99f;       // 4 bytes
double salary = 50000.50;   // 8 bytes
char grade = 'A';           // Single character
boolean isActive = true;    // true or false
```

## Common Methods

```java
String name = "Danish Khan";
name.length();              // 12
name.toUpperCase();         // DANISH KHAN
name.substring(0, 6);       // Danish
name.contains("Khan");       // true
```

## Key Takeaways

- Java is platform-independent
- Understand data types and control flow
- String operations are essential
- Arrays and collections are frequently used
